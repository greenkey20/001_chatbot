Indexing Pipeline

    Data Loading: JSON 파일로부터 FAQ 문서를 읽어 Document 객체 생성
  • ●
    Embedding: 선택된 모델(OpenAI/HF/Ollama)을 통한 벡터화
  • ●
    Storage: Chroma DB에 메타데이터(keyword, question_id)와 함께 저장

  • ●
    Metadata Filter: LLM이 질문을 분석하여 Chroma 필터 조건 자동 생성
  • ●
    Persistence: SQLite에 대화 저장 및 임계값 도달 시 자동 요약 압축
  • ●
    Evaluation: 생성된 답변의 품질을 LLM이 자가 진단하여 신뢰성 확보
